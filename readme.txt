Скрипт, который находит все личные сообщения с указанным словом (или словосочетанием) и составляет диаграмму частоты его использования (топ-10 диалогов).

Ajax Listener via http://qaru.site/
Круговая диаграмма на canvas via https://code.tutsplus.com

:: ИНСТРУКЦИЯ ПО ЗАПУСКУ ::

1. Перейдите на страницу с диалогами: https://vk.com/im
2. Откройте браузерную консоль (Ctrl + Shift + I). Скопируйте содержимое файла pm_stats.js в браузерную консоль и нажмите Enter.
3. В поле ввода начните вводить слово (или словосочетание) для поиска. Нажмите Enter. Дождитесь загрузки.
3. Enjoy!
